# Statement Financial Model

A dynamic 3-statement financial model built in Microsoft Excel that integrates the Income Statement, Balance Sheet, and Cash Flow Statement.

## Features
- Linked financial statements using formulas
- Automatic updates based on input assumptions
- EBITDA, Working Capital, and Free Cash Flow calculations
- Clean and structured financial model

## Tools Used
- Microsoft Excel

## Purpose
Designed for financial analysis, forecasting, and understanding the relationship between core financial statements.
